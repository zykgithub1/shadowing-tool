如何获得最好的语音质量？

推荐：在线AI语音（需联网）
使用已部署的 Cloudflare Worker（Edge 神经网络 TTS），无需登录第三方账号。若 Worker 上设置了 API Key，请在页面脚本里填写 TTS_API_KEY。

备选：优化本地语音包
Android: 设置 → 无障碍 → 文字转语音(TTS)输出，选Google引擎，下载高质量英文语音包。
iPhone: 系统自带英文语音质量不错，无需额外设置。

进阶：安装微软语音引擎
GitHub搜索 tts-server-android，安装后系统设置切换默认TTS引擎，可获得微软Edge同款语音（需联网）。
