# 🎧 ShadowEcho - 影子跟读练习器

免费开源的英语影子跟读工具。粘贴英文文本，逐句朗读并停顿，等你跟读。

## 功能

- **逐句朗读**：自动按句拆分文本，一句一句朗读
- **停顿跟读**：每句朗读后自动停顿，留时间给你跟读
- **在线AI语音**：接入 Puter.js 免费AI语音，质量接近真人（需联网，首次需登录）
- **本地语音备用**：使用浏览器内置语音引擎，离线可用
- **灵活控制**：可调语速、停顿时长、每句重复次数
- **点击跳转**：可点击任意句子跳转到该句
- **手机友好**：移动端优先设计，适合手机使用

## 使用方法

### 在线使用（推荐）

访问：`https://zykgithub1.github.io/shadowing-tool/`

### 自己部署

1. Fork 这个仓库
2. 进入仓库 Settings → Pages
3. Source 选 `Deploy from a branch`，Branch 选 `main`，文件夹选 `/ (root)`
4. 保存，等1-2分钟即可通过 GitHub Pages 访问

### 本地使用

直接用浏览器打开 `index.html` 即可（本地模式下在线AI语音不可用，会自动切换到本地语音）

## 语音质量优化

### 方案一：在线AI语音（推荐）
- 工具默认使用 Puter.js 提供的免费AI语音
- 首次使用会弹出登录框，支持 Google 一键登录
- 登录一次后浏览器会记住，不会再弹

### 方案二：优化手机本地语音包
- **Android**：设置 → 无障碍 → 文字转语音(TTS)输出 → 选择 Google 引擎 → 下载高质量英文语音包
- **iPhone**：系统自带英文语音质量较好，无需额外设置

### 方案三：安装微软语音引擎（Android）
- GitHub 搜索 `tts-server-android`，安装后在系统设置切换默认TTS引擎
- 可获得微软 Edge 同款高质量语音（需联网）

## 技术栈

- 纯 HTML/CSS/JS，单文件，零构建依赖
- [Puter.js](https://puter.com) - 免费在线AI语音
- Web Speech API - 浏览器本地语音

- 影子跟读（Shadowing）练习

- 任何需要逐句跟读的英语学习场景
